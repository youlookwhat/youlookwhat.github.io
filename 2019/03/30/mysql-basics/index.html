数据库操作

;是结束符

  • 1.创建数据库
    create database mysql_0219;

  • 2.查看所有数据库
    show databases;

  • 3.使用数据库
    use mysql_0219;

  • 4.删除数据库
    drop database mysql_0219;

  • 5.查看数据库状态
    status (注意编码：utf-8)

表的创建与管理:

  • 1.创建表
    create table t_user1(id int,name varchar(20),age int,birthday date);

  • 2.修改表
    alter table t_user1 rename t_user2;

  • 3.查看具体表
    desc t_user2;

  • 4.查看所有表
    show tables;

  • 5.删除
    drop table t_user2;
