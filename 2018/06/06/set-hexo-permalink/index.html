永久链接

因为title可能为中文，那生成的链接就可能为上面的情况，这时候我们希望自己在每篇文章的Front-matter中指定一个字段，然后替换title，这样就可以显示我们自己定义的url了，例如：

  • permalink: :year/:month/:day/:urlname/
  • permalink: post/:urlname/
